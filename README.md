Dependencies
    Ruby 2.6.1
    Rails 6.1.1
    Json Web Token (JWT)
    PostgreSQL
    Bcrypt

Installation

    Clone this project

      $ git clone https://github.com/BernarBerdikul/rails_jwt.git

    Change directory to this project

      $ cd rails_jwt

    Install dependencies

      $ bundle install

    Create database

      $ rails db:create

    Run migration

      $ rails db:migrate

    Run this project

      $ rails server
